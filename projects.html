Broca-translator

Seq2Seq model for translation of (Broca) aphasic sentences into English. Project developed as part of the course of Natural Language Processing at the Reykjavik University.

Malware Classification with Cuckoo Sandbox

This project showcases the implementation of a malware classification project utilizing the Cuckoo sandbox. We collected diverse malware samples from various GitHub repositories, subjected them to analysis using the Cuckoo Sandbox, and explored a method for classifying them based on a defined set of features. This project served as the Final Project for the MLCS (Machine Learning in Cyber Security) course at Reykjavik University.

Network Intrusion Detection

Conducted thorough data preprocessing, including removal of leading/trailing spaces from column names and values, handling ”Infinity” and "NaN" values by replacing them with appropriate numerical limits. Implemented Decision Tree and Random Forest classifiers, achieving an accuracy of 99.87% and 81.79%, respectively. Utilized scikit-learn for metrics and pandas for preprocessing in model evaluation. PyCharm and Jupyter are used as IDE and file format.

JanusGraph-API

Simple REST API implementation to query an instance of JanusGraph database, using Cassandra as backend storage. Developed as project of Technologies for Big Data Management exam in the Computer Science degree of University of Camerino.

Algorithms and Data Structures

Entirely developed Java library representing all the major graph researching algorithms like Dijkstra, Bellman-Ford, etc., and all the major data structures used to represent graphs over memory, like binary heaps or adjacency lists. Project developed as part of Algorithms and Data Structures course at UNICAM

Image Cataloging Support

Group project started at UNICAM as a tool that aims to recognize faces and stamps inside of historical images. This tool makes use of Single Shot MUltibox Detector algorithm for objects recognition, and it is further trained over the dataset given by a local museum in Italy.
